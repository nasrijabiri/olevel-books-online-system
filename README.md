# Project title:  olevel-books-online-system.
# Project Description: A web-based system for purchasing O-Level books online. This project aims to provide an online platform where O- Level students can search, view, and purchase textbooks online.
# Objectives:                                                                                                                                   a) Allow students to search for books.                                                                                                     b) Allow students to view book details.                                                                                                    c) Allow students to place orders online.                                                                                                  e) Enable administrators to manage books.                                                                                                  f) Enable administrators to manage orders.
# Technologies used are:                                                                                                                        a) HTML                                                                                                                                    b) CSS                                                                                                                                     c) JavaScript                                                                                                                              d) PHP                                                                                                                                     e) MySQL
