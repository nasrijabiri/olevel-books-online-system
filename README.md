# olevel-books-online-system
# A web-based system for purchasing O-Level books online
# Project Description:                                                                                                                     This project aims to provide an online platform where O-Level students can search, view, and purchase textbooks online.
# Objectives                                                                                                                                    a) Search books                                                                                                                            b) View book details                                                                                                                       c) Purchase books online                                                                                                                   e) Manage books                                                                                                                            f) Manage orders
# Technologies used are;                                                                                                                        a) HTML                                                                                                                                    b) CSS                                                                                                                                     c) JavaScript                                                                                                                              d) PHP                                                                                                                                     e) MySQL
